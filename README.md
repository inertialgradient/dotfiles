dotfiles
=========

Installation
------------

1. Install Xcode & Dropbox

2. Clone this repo

   ```shell
    git clone --recurse-submodules -j8 https://github.com/inertialgradient/dotfiles ~/.dotfiles
   ```

3. Run the bootstrap script

   ```shell
    ~/.dotfiles/bin/bootstrap
   ```

   Stick around to enter your password as needed.
